# Solar-Powered Smart Sprinkler System

## Overview

The Solar-Powered Smart Sprinkler System is a prototype irrigation solution designed to reduce water wastage through automated soil moisture monitoring.

---

## Problem Statement

Traditional irrigation methods often result in:

- Excessive water consumption
- Energy wastage
- High labour requirements
- Reduced irrigation efficiency

---

## What We Built

- Soil moisture sensing
- Arduino-based control
- Automatic sprinkler activation
- Solar-powered operation

---
